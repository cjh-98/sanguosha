## 一、严重问题（必须修复）

### 1. 五谷丰登 - 实现完全错误 ❌
**位置**: game.js 第882-898行  
**问题描述**: 
- 规则：使用五谷丰登时，从牌堆亮出等同于存活角色数的牌，所有角色按行动顺序依次选择其中一张获得
- 当前实现：随机分配牌给每个角色，没有选择过程
- **影响**: 完全改变了卡牌效果，严重影响游戏平衡性

**错误代码**:
```javascript
case 'wugu': // 五谷丰登
    const aliveCount = this.getAlivePlayers().length;
    const wuguCards = [];
    for (let i = 0; i < aliveCount; i++) {
        const c = this.revealTopCard();
        if (c) wuguCards.push(c);
    }
    // 简化：每人获得一张
    for (const p of this.getAlivePlayers()) {
        if (wuguCards.length > 0) {
            const c = wuguCards.pop();  // ❌ 随机分配，不是选择
            p.handCards.push(c);
        }
    }
    break;
```

**修复建议**: 需要实现UI交互，让玩家依次选择想要的牌
