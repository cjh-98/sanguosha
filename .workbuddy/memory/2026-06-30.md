# 2026-06-30 工作日志

## 修复的关键bug

### 1. 洛神红色牌不自动退出（严重）
- **根因**：`finishLuoshen()` 调用 `stepTurn()` 导致 `doBegin()` 被重新执行，洛神技能再次触发
- **修复**：改为调用 `advancePhase()` 直接推进到判定阶段

### 2. 游戏流程不串行（严重）
- **根因**：`doJudge()` 不检查等待状态就直接推进阶段
- **修复**：添加等待状态检查（`_waitingForGuicai`, `_waitingForGuidao`）

### 3. `_continueJudgeAfterSkill` 严重bug（严重）
- **根因1**：函数中错误调用 `this.revealTopCard()` 来比较卡牌（会错误地从牌堆弹出一张牌！）
- **根因2**：函数中使用 `this._guicaiPlayer || this._guidaoPlayer` 获取player，但在 `finishGuicai/finishGuidao` 中这些变量已被设为null
- **修复**：移除错误调用；在调用前保存player引用并传入

### 4. 洛神取消按钮bug（严重）
- **根因**：取消按钮只关闭弹窗，没有调用 `finishLuoshen()`，导致 `_waitingForLuoshen` 永远为true，游戏永久阻塞
- **修复**：添加 `cancelLuoshen()` 函数正确处理取消

### 5. 鬼才/鬼道取消按钮bug（严重）
- **根因**：同上，取消按钮只关闭弹窗
- **修复**：添加 `cancelGuicai()` 和 `cancelGuidao()` 函数
